## Usage
1. clone CosmoLady repository
   ```shell
   git clone https://github.com/leeyoshinari/Small_Tool.git
   
   cd Small_Tool/CosmoLady
   ```

2. set the MySQL's `username` and `password`, etc.

3. run
   ```shell
   python SpiderBar.py
   ```
   
## Requirements
1. requests
2. pymysql
